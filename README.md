# Oasis Infobyte Web Development Internship

This repository contains the projects completed during my Web Development Internship at **Oasis Infobyte (OIBSIP)**. These tasks demonstrate proficiency in Frontend technologies, including responsive design, security validation, DOM manipulation, and modern UI/UX principles.

---

## 🚀 Projects Overview

### 1. Task_01: AeroCalculator
A specialized, high-performance calculator themed with an aerospace aesthetic.
* **Core Technologies:** HTML5, CSS3, JavaScript (ES6)
* **Key Features:** Handles complex arithmetic and percentages with a clean, high-contrast interface.
* **Technical Highlights:** Use of CSS Grid for layout and event listeners for real-time calculation.
* **🔗 Live Demo:** [View AeroCalculator](https://eswarchandra999.github.io/OIBSIP_Web_Development/Task_01_AeroCalculator/)

### 2. Task_02: Tribute Page
A professional tribute page dedicated to **Dr. APJ Abdul Kalam**, the Missile Man of India.
* **Core Technologies:** HTML5, CSS3
* **Key Features:** Semantic HTML structure, professional branding, and responsive image containers.
* **Technical Highlights:** Focus on visual hierarchy and high-quality image integration without background clutter.
* **🔗 Live Demo:** [View Tribute Page](https://eswarchandra999.github.io/OIBSIP_Web_Development/Task_02_TributePage/)

### 3. Task_03: Agenda Manager
A functional To-Do application built for daily productivity and task management.
* **Core Technologies:** HTML5, CSS3, JavaScript (Local Storage)
* **Key Features:** Data persistence using LocalStorage and dynamic DOM manipulation for task tracking.
* **Technical Highlights:** JSON parsing for data storage and dynamic element creation/removal.
* **🔗 Live Demo:** [View Agenda Manager](https://eswarchandra999.github.io/OIBSIP_Web_Development/Task_03_Agenda/)

### 4. Task_04: Authentica Portal
A modern, high-fidelity authentication portal featuring real-time security validation.
* **Core Technologies:** HTML5, CSS3 (Glassmorphism), JavaScript (Regex)
* **Key Features:** Real-time password tracking (Uppercase, Numbers, Special Characters) and responsive UI.
* **Technical Highlights:** Advanced Regex for input validation and CSS backdrop-filter for glass effects.
* **🔗 Live Demo:** [View Authentica Portal](https://eswarchandra999.github.io/OIBSIP_Web_Development/Task_04_Authentica/)
