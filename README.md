# Oasis Infobyte Web Development Internship

This repository contains the projects completed during my Web Development Internship at **Oasis Infobyte (OIBSIP)**. These tasks demonstrate proficiency in Frontend technologies, including responsive design, DOM manipulation, and modern UI/UX principles.

---

## 🚀 Projects Overview

### 1. Task_01_AeroCalculator
A specialized, high-performance calculator themed with an aerospace aesthetic. 
*   **Core Technologies:** HTML5, CSS3, JavaScript (ES6).
*   **Key Features:**
    *   **Logic:** Handles complex arithmetic operations including basic math and percentages.
    *   **UI/UX:** Clean, isolated interface with high-contrast buttons for better readability.
    *   **Responsiveness:** Fully adaptive layout that works on mobile and desktop browsers.
*   **Technical Highlights:** Use of CSS Grid/Flexbox for button alignment and event listeners for real-time calculation updates.

### 2. Task_02_TributePage
A professional tribute page dedicated to **Dr. APJ Abdul Kalam**, the Missile Man of India.
*   **Core Technologies:** HTML5, CSS3.
*   **Key Features:**
    *   **Branding:** Clean typography and professional color palette.
    *   **Interactive Elements:** Styled image containers with hover effects and accessible text content.
    *   **Structure:** Semantic HTML tags used for better SEO and screen-reader accessibility.
*   **Technical Highlights:** Focus on visual hierarchy and high-quality image integration without background clutter.

### 3. Task_03_Agenda
A functional "Agenda Manager" or To-Do application for daily productivity.
*   **Core Technologies:** HTML5, CSS3, JavaScript (Local Storage).
*   **Key Features:**
    *   **CRUD Operations:** Users can Create, Read, and Delete tasks.
    *   **Data Persistence:** Integrated with **LocalStorage** so your tasks remain even after refreshing the browser.
    *   **Interface:** User-friendly input fields and dynamic task lists.
*   **Technical Highlights:** DOM manipulation for adding/removing elements dynamically and JSON parsing for data storage.
